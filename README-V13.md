# V13 National Showcase Edition

เว็บไซต์ศูนย์ศึกษาพระพุทธศาสนาวันอาทิตย์ วัดสี่แยกสมเด็จ  
ฉบับเตรียมนำเสนอผลงานระดับประเทศ

## สิ่งที่เพิ่ม

- ค้นหากิจกรรมและรางวัลจากหน้าเว็บไซต์
- กรองรายการตามปี
- ปุ่มดูเพิ่มเติม ลดความยาวหน้าเว็บ
- เรียงรางวัลจากปีล่าสุด
- ช่องโทรศัพท์และ Facebook ในหลังบ้าน
- ปุ่มโทรศัพท์ เปิด Facebook และแชร์เว็บไซต์
- Meta Description และ Open Graph สำหรับการแชร์
- ปรับคำอธิบายภาพ ปุ่ม และหน้าต่างให้รองรับการเข้าถึง
- รองรับผู้ใช้ที่ตั้งค่าลดภาพเคลื่อนไหว
- จัดลำดับอัลบั้ม บุคลากร และรางวัลด้วยปุ่มขึ้น–ลง
- ตั้งชื่อโฟลเดอร์อัลบั้มใหม่ตามรูปแบบ `ปี_ชื่ออัลบั้ม`
- เครื่องมือจัดชื่อโฟลเดอร์เดิมโดยไม่ลบหรือย้ายรูปภาพ
- สำรองอัตโนมัติ กู้คืน 10 ชุด และ Activity Log
- สถานะฉบับร่าง/เผยแพร่
- ตรวจไฟล์ภาพซ้ำก่อนอัปโหลด

## การติดตั้งโดยไม่ให้รูปภาพเดิมหาย

1. ใช้ Google Apps Script Project เดิม
2. นำ `gas/Code.gs` ไปแทนโค้ดเดิม
3. Deploy เป็นเวอร์ชันใหม่ โดยใช้ Spreadsheet และ Drive Folder เดิม
4. ใช้ค่า `VITE_GAS_API_URL` เดิมก่อน Build/Deploy หน้าเว็บ
5. ห้ามรัน `setup()` เพื่อสร้างฐานใหม่ หากระบบเดิมมีค่า Script Properties ครบแล้ว

รูปภาพเดิมเก็บอยู่ใน Google Drive และไม่ถูกบรรจุไว้ใน ZIP นี้ การอัปเดตโค้ดจึงไม่ลบรูปภาพเดิม

## การจัดชื่อโฟลเดอร์เดิม

หลังติดตั้ง Code.gs เวอร์ชันนี้ ให้เข้าหน้า “สำรองและประวัติ” แล้วกด
“จัดชื่อโฟลเดอร์” ระบบจะเปลี่ยนเฉพาะชื่อโฟลเดอร์แบบเดิม
`อัลบั้ม_<รหัส>` เป็น `ปี_ชื่ออัลบั้ม` โดยไม่ลบหรือย้ายรูปภาพ

## การตรวจสอบ

- `npm run build` ผ่าน
- ตรวจไวยากรณ์ Google Apps Script ผ่าน
- ZIP ไม่รวม `.env`
